# Bracketians!
document stuff with it.

file format: `.bm` Bracketian Markdown

## Basics:
```nim
call []
list [!] => [! 1 2 3 4 5]
list () => (1 2 3 4 5)
table [:] => [: name "ali" age 12]

symbol fa
text "dasd"
number 3213 12.221
```
